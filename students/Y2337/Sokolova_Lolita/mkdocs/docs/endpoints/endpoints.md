# Endpoints

## ``` /drivers/ ```

Доступные методы:

* get - получить список водителей
* post - создать водителя

## ``` /drivers/:id/ ```
Доступные методы:

* get - получить водитея
* put - изменить водитея
* patch - изменить водитея
* delete - удалить водитея
