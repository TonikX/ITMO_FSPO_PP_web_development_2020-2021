	1. Frontend и Backend

	Frontend - все то что может осуществить браузер, то есть читать выводить на экран или запускать. Другими словами это клиентская сторона пользовательского интерфейса. 
	Backend - программно-аппаратная часть сервера, которая представляет собой набор средств с помощью которых происходит реализация логики веб-сайта, то есть реализовывается то как работает тот или иной элемент веб-сайта. 

	2. Примеры Frontend и Backend феймворков. 

	Самые популярные бэкенд-фреймворки:
1)Django (язык програмирования Python);
2)Symfony (язык программирования PHP);
3) Express.js (язык программирования JavaScript);
4)Ruby on Rails (язык программирования Ruby);
	Фронтенд-фреймворки:
1)Angular (язык программирования JavaScript);
2)Vue.js (язык программирования JavaScript);
3)Svelte (язык программирования JavaScript);

	3.Отличие Frontend и Backend:

Frontend	
Основной язык фронтенда является JavaScript
Широко используются фронтенда-разработки: 
	-UI-фреймворки вроде React или Vue,
	-веб-фреймворки, такие как Gatsby,
	-компиляторы (Babel),
	-сборщик (Wabpack),
	-CSS-инструменты, например Sass. 

Backend
Основные языки программирования для бэкенда: PHP, Python, JavaScript, Java, .NET(VB и C#), Go, C#. 
Распространение инструменты бэкенда:
	-облачные серверы  вроде AWS EC2,
	-бессерверные сервисы, такие как AWS Lambda,
	-базы данных NoSQL (MongoDB),
	-такие языки как Python или JavaScript (последний — с применением NodeJS),
	-фреймворки веб-приложений, например Serverless Framework.
