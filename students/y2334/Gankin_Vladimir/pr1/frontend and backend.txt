Frontend - часть которую видит пользователь и с которой он может взамодействовать,
создается средствами HTML, CSS и чаще всего JavaScript.

Frontend фреймворки:
React.js - JavaScript-библиотека для создания пользовательских интерфейсов,
разрабатываются отдельные компоненты, которые модно объединять в сложные
пользовательские интерфейсы.
Angular - JavaScript и TypeScript-фреймворк для создания клиентских приложений. Направлен на разработку
одностраничных приложений.
Vue.js - JavaScript фреймворк для создания пользовательских интерфейсов. В отличие от фреймворков-монолитов, 
Vue создан пригодным для постепенного внедрения.

Backend - серверная часть сайта, обрабатыет запросы пользовтеля.

Backend фреймворки:
Laravel - бэкенд-фреймворк для решения однотипных задач. Его можно применить в процессе разработки 
приложений различных типов, если для них актуально использование PHP. Широко реализованы возможности для 
организации бэкендов в мобильном софте.
Flask - установка этой основы даёт вебмастеру доступ к шаблонизатору и маршрутизации запросов. Предполагается, 
что оставшиеся элементы будут разработаны собственноручно, либо подключены из числа имеющихся.
Express.js - Это гибкий веб-фреймворк, являющийся наиболее популярным при создании Node.js приложений. Данное техническое 
решение актуально для мобильного софта и веб-приложений. В распоряжении разработчика оказываются фундаментальные
 возможности для работы с функционалом Node.js. Можно пользоваться реляционной и нереляционной БД, подключать 
всевозможные верификационные модули и надстройки для форматирования данных.
