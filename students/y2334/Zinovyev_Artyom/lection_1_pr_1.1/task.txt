Backend:
    Серверная часть проекта/ресурса.
    Внтренняя логика.
    Обработка запросов от клиента, включая отдачу фронт-енд части.
Frontend:
    Пользовательский интерфейс, логика обращения к серверной части приложения (к backend-у)

Frontend frameworks:
    React:
        Файлы в формате .jsx, свой синтаксический сахар
    Angular:
        Код на typescript

Backend frameworks:
    Express.js - Фреймворк для JS, поддерживает пользовательские расширения
    Ruby on rails - Фреймвор для Ruby, умирает
    Django - Фреймворк для Python с сильнейшей ORM, которую когда-либо видел Антон Игоревич
