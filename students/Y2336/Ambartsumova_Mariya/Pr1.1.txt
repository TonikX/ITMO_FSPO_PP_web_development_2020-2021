	Frontend — это разработка пользовательского интерфейса и функциональности, 
которые работают на клиентской стороне веб-сайта или приложения. 
	К этому виду разработки можно отнести все, что видит пользователь, открывая web-страницу. 
	К этому виду разработке относятся:
	- HTML -язык разметки документов, при помощи которого формируется структура страницы
	- CSS - язык для описания и стилизации внешнего вида документа
	- JavaScript - язык, который должен реагировать на действия пользователя, обрабатывать клики мышкой, 
перемещения курсора, нажатия клавиш.
	Примеры фреймворков: Bootstrap, Semantic UI, Foundation.

	Backend-разработка реализует логику работы сайта, делает то, что скрыто от глаз пользователей.
	Для backend-разработи применяются любые инструменты, доступные на его сервере. Можно выбрать любой из 
универсальных языков программирования, например, Ruby, PHP, Python, Java. Также для backend-разработки используются 
разные СУБД.
	Примеры фреймворков:(Django, Plone,Ruby on Rails, Spring MVC.

	Различия:
	- frontend разработка видна пользователя, backend - нет;
	- frontend разработка ограничена в выборе средств реализации;
	- при разработке backend-а можно работать с БД.
