Фронтенд – это клиентская часть сайта или любого веб-ресурса. Фронтенд представляет собой пользовательский интерфейс для взаимодействия с веб-сервисом. Это то, что видит пользователь при загрузке сайта и некоторые возможности при взаимодействии с ним (например, код на javaScript, который оживляет страницу, добавляя некий интерактив на сайт). 
Для фронтенд разработки используется HTML (язык разметки гипертекста), CSS (таблица стилей, позволяющая красиво или не очень оформить страницу) и различные инструменты для дизайна. 
Бэкенд предназначен для обеспечения взаимодействия между пользователем и сервером. Эта часть веб-разработки, которая отвечает за внутреннюю логику сервиса. Эту часть пользователь не видит, но она позволяет пользователю взаимодействовать с функионалом сервиса. 
Наиболее популярные фреймворки для фронтенда:
Один из самых известных CSS-фреймворков – Boostrap. Имеет в своем составе шаблоны для отрисовки кнопок, навигационных панелей, форм и других элементов сайта. Включает в себя JavaScript-расширения. 
Skeleton – фреймворк для создания адаптивных сайтов, то есть тех, которые адаптируются к формату устройства, на котором открыт веб-сервис. 
Фреймворки для бэкенда делятся по принадлежности к языку программирования для бэкенда.
Один из самых популярных – Django, он для языка на Python. Аутентификация, обмен сообщениями, маршрутизация, работа с базой данных, админская часть сайта – работу с этими частями упрощает данный фреймворк. Фреймворк реализует многие важные принципы защиты самостоятельно, например, предотвращает выполнение кода на уровне шаблонов.
Фреймворк для PHP – Laravel довольно молод, но уже достаточно популярен. Многие возможности, например поддержка API, доступны из коробки.
